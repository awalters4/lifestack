# LifeStack

> So here’s the deal:  
> I’ve been “unemployed” for almost three years.  
> Had a baby in that time — she’s almost two now.  
> And guess what? It’s time. Time to get back to work.

**LifeStack** is a tool I’m building to make life easier — for me, and eventually, for others.  
It’s designed to help busy people (especially parents, entrepreneurs, and creatives) track routines,  
build habits, and organize their days with intention. Think of it like a digital command center  
for your life — built by someone who’s living it in real time.

This project is my way of saying:  
**I’m back, baby.**

---

### 🛠 Tech Stack (for now)

- **React** (with TypeScript) — frontend framework
- **Tailwind CSS** — utility-first styling
- **Framer Motion** — smooth animations and transitions
- **OAuth** — secure user authentication
- **Supabase** — real-time backend (auth + database)

---
